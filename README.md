# heyry-schemas
Public registry of all HEYRY Tools JSON Schemas. Defines the canonical data contracts for policies, procedures, specifications, products, IoT configs, QR targets, and core entities. Includes versioned schemas, reusable building blocks, governance meta-schema, CI validation, and a machine-readable index.
